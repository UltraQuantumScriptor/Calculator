###### SCIENTIFIC CALCULATOR

## What it is:

This scientific calculator is a recreation of the calculator, Casio fx-82MS in python, with stripped down functions(square roots, separate exponenet buttons, etc)

## How to use it:

Upon opening, you can type any expression and it will output the result.
type "exit" to exit the program
type "guide" to print a basic guide on all the functions currently present in the calculator
Scientific notations are written like this: 5 x10 12(5e+12) where 5 is the mantissa and 12 is the exponent
